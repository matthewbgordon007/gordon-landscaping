Interlock Cost in Ottawa: What Impacts Price

Interlock pricing isn’t just “cost per square foot.” The real cost comes from base work, access, and design details. Here’s the clean breakdown.

1) Excavation + disposal

Removing old material and digging to spec is a major cost driver, especially if disposal is required.

2) Base depth + compaction

This is the foundation. More depth and proper compaction reduce settling and freeze/thaw movement.

3) Access

If we can machine-access the yard, it’s faster. If everything is wheelbarrowed by hand, labour increases.

4) Stone choice + pattern

Premium stone and complex patterns increase material cost and install time.

5) Edges, steps, and transitions

Borders, soldier courses, steps, and clean transitions into turf/sod all add time and detail work.
